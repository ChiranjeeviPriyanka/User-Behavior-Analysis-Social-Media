# User-Behavior-Analysis-Social-Media

A data analytics and machine learning project analyzing social media user behavior using clustering algorithms (K-Means, DBSCAN) to identify engagement patterns and optimize user interactions.

## Project Overview

This project analyzes user behavior across multiple social media platforms to understand engagement patterns. It includes data preprocessing, statistical analysis, and clustering to segment users and provide insights for personalized content and improved engagement.

**Dataset:** [Social Media Usage and Emotional Well-Being (Kaggle)](https://www.kaggle.com/datasets/emirhanai/social-media-usage-and-emotional-well-being/data)
